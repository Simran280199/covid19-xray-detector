# 🫁 COVID-19 X-Ray AI Detector

Deep Learning app to detect COVID-19 from chest X-Ray images.

## Models
- CNN Basic — 84.85% accuracy
- ResNet50 — 80.30% accuracy

## Features
- Single image prediction
- EDA Dashboard
- Model comparison
- Batch prediction + CSV export

## Deploy on Streamlit Cloud
1. Push this repo to GitHub
2. Go to share.streamlit.io
3. Connect your GitHub repo
4. Add Kaggle secrets
5. Deploy!
